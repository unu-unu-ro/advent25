# Icons Directory

Place your icon files here with the following sizes:
- icon-72.png (72x72)
- icon-96.png (96x96)
- icon-128.png (128x128)
- icon-144.png (144x144)
- icon-152.png (152x152)
- icon-192.png (192x192)
- icon-384.png (384x384)
- icon-512.png (512x512)

Also place:
- favicon.ico (32x32)
- apple-touch-icon.png (180x180)
- og-image.jpg or og-image.png (1200x630 for social sharing)

Recommended design:
- Dark teal/green background (#1a2928)
- Gold star or cross symbol (#D4A850)
- Number "25" or Advent candle motif
- Clean, minimal design that works at all sizes
